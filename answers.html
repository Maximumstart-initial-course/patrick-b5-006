Что такое и зачем нужен HTML?

HTML-это язык разметки.

Что такое и зачем нужен Git?

Git-это система контролий версий. Она позволяет разрабатывать один проект сразу несколькими разработчиками.

Что такое SEO?

SEO-это меры для поднятия сайта в поиске.

В чем отличие Git от GitHub?

Git-программа, а GitHub-сайт.

Что такое тег? Что такое атрибут?

Тег-элемент языка разметки. Атрибут-часть тега.

Чем отличается язык разметки от языка програмирования? HTML можно назвать языком программирования?

Язык программирования нужен для создания программ, а язык разметки для создания макетов и веб-шаблонов. Нет, HTML нельзя назвать языком программирования, он предназначен для другого.

Зачем нужен атрибут alt у изображений?

Он нужен для того, что-бы пользователь при ошибке или не загрузке изображения получил информацию о нём.

Зачем нужен тег <label>? Как объеденить <label> и <input>?

Устанавливает связь между определенной меткой. У <label> for должно быть такое же значиние как у id <input>.

Как объеденить радиокнопки в группу?

Всем радиокнопкам нужно задать одинаковый атрибут name.

Какие условия нужно соблюсти, чтобы отправить файл на сервер?

Нужен method="POST", и нужно указать сайт в action, что-бы отправлять файлы в форме нужен enctype который будет равен multipart/...

Что такое HTTP? Какие существуют HTTP методы?

HTTP-протокол передачи данных. POST, GET, PUT, DELETE.

Зачем используется атрибут name?

Name нужен для отправки данных на сервер.

Что такое семантика? Зачем нужна?

Cемантика - организация структуры сайта. Семантика нужна для улучшения структуры веб-страницы.

В каких случаях используется тег <article&ld;?

При наличии новостей, статьи, записи блога, форума

Зачем используются ARIA-роли и ARIA-атрибуты?

Их используют для того, что-бы сделать сайт удобнее для людей с ограниченными способностями.

Зачем используются теги <iframe>, <audio>, <video>?

<video>/<audio>-добавляет и воспроизводит видеоролик/аудио на сайте. <iframe>-контейнер, который позволяет загружать в область заданных размеров любые другие независимые документы.

Какие хорошие и плохие стороны формата JPEG?

Хорошая - Поддерживает более 16 миллионов цветов. Плохая - JPEG сжимает изображения с потерей качества.

Какие отличия PNG-8 от PNG-24?

PNG-8-используем, когда работаем с изображениями с небольшим количеством цветов.PNG-24-используем, когда необходима повышенная точность полноцветных изображений

Для чего может использоваться формат GIF?

Особенностью GIF является поддержка анимации.

Что такое векторная графика? Какие ее преимущества перед растровой?

Векторная графика-используется для иллюстраций, иконок и логотипов. В отличие от растровых форматов, можно взаимодействовать с изображениями в формате SVG при помощи CSS

Для чего используется сервис Tinypng и подобные ему?

Что-бы оптимизировать изображение(что-бы оно меньше весило).

В чем преимущества Avocode по сравнению с Photoshop?

Avocode проще в освоении, и у него удобнее интерфейс.

За что отвечает свойство display?

display позволяет сменить тип элемента (flex, grid...).

Чем отличается width: 100% от width: auto?

width: 100% - занимает всю ширину родительского блока, width: auto - растягивает елемент на ширину своего внутреннего содержимого.

Зачем нужно свойство box-sizing?

box-sizing изменяет алгоритм расчета ширины элемента, при смене он включает в себя padding и border.

Чем отличаются свойства justify-content и align-items?

justify-content задает положение элементов вдоль главной оси, a align-items на поперечной оси.

Как заставить флексовые элементы переноситься на новую строку?

Для этого нужно использовать свойство flex-wrap.

Назови два способа выравнивания: 1) блочного элемента по центру; 2) строчного элемента по центру.

1) margin: 0 auto; 2) text-align: center;

Чем отличается абсолютное позиционирование от относительного?

Разница лишь в отправной точке: в положении точки нулевых координат.

Что такое трек сетки в CSS Grid?

Трек - это расстояние между линиями в CSS Grid.

Для чего используются свойства grid-area и grid-template-area?

grid-area дает название элементу, а grid-template-areas определяет шаблон сетки ссылаясь на имена областей

Что такое единица fr?

fr - это новая форма единицы измерения, часто используется в CSS Grid.

Зачем используются и как работают grid-column/grid-row?

Это сокращенные записи другий свойств, grid-column-вертикальные линии, grid-row-горизонтальные линии.

Что такое Flexbox и CSS Grid?

Flexbox - набор свойств, которые помогают нам строить сетки, CSS Grid-это новая технология которая позволяет удобнее строить сетки.

Как строить сетки на флоатах (float)? Для чего должно использоваться свойство float?

Определяет, по какой стороне будет выравниваться элемент, другие элементы будут его обтекать с другой стороны. Его использовали раньше, когда ещё не было Grid CSS.

Чем отличается Flexbox свойство justify-content от align-items?

justify-content задает положение элементов вдоль главной оси, a align-items на поперечной оси.

Зачем используется Flexbox свойство flex-basis?

flex-basis ‒ определяет базовый размер элемента

Чем отличается flex-grow от flex-shrink?

flex-grow - определяет, то, насколько отдельный flex-блок может быть больше соседних элементов. flex-shrink ‒ определяет, насколько flex-блок будет уменьшаться относительно соседних элементов внутри.

Чем отличаются CSS Grid свойства justify-items, justify-content и justify-self?

justify-items-выравнивает содержимое, justify-content - выравнивает контейнер, justify-self-выравнивает содержимое одного элемента.

Зачем используется свойство grid-area?

grid-area дает название элементу, что-бы можно было ссылаться на него с помощью шаблона созданого через grid-template-area.

Сокращением от каких свойств является свойство grid-gap? Сколько значений они принимают?

grid-column-gap и grid-row-gap, принимают 2 значения.
